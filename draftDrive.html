Process Analysis

"Draft Drive" Backend Website



The Draft Drive platform is a web-based application made with PHP, CSS3, JavaScript and MySQL designed to immerse basketball enthusiasts in the world of the National Basketball Association (NBA). The site provides a rich user experience by offering a plethora of features such as player comparisons, team insights, personalized dream teams, and real-time data interaction. Below is a flowchart of the database design.

I designed a robust relational database using MySQL to support the complex interactions within the Draft Drive platform. The database consists of interconnected tables for Users, Teams, Players, and Dream Teams, structured to maintain data integrity and efficiency. It utilizes foreign key relationships, such as linking team data to players and dream teams to users, ensuring seamless data retrieval. This structure allows users to create and save dream teams, add ratings and comments, and explore comprehensive player statistics and team details dynamically. The database design also facilitates easy updates and scalability to accommodate additional features and future growth.

I then implemented an engaging and responsive front-end using HTML, CSS, and JavaScript, ensuring compatibility across a wide range of devices and screen sizes. The user interface features intuitive design elements like dropdowns, cards, and scrollable containers for browsing teams and players. On the back-end, I used PHP to handle core functionality, such as user authentication, secure session management, and dynamic database queries. AJAX functionality was integrated to enable real-time data updates without reloading pages, providing a smooth and interactive user experience, especially for features like player comparisons and team selections.
